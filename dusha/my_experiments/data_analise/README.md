# Audio and Text Analysis Scripts

Скрипты для анализа аудио и текстовых признаков, визуализация данных.

## Оглавление

1. [text_analise.py](#text_analise) - Анализ текстов: TF-IDF, важные слова, биграммы
2. [audio_analise.py](#audio_analise) - Анализ аудио: mel-спектрограммы, MFCC, PCA, t-SNE
3. [main_statistic.py](#main_statistic) - Статистический анализ датасетов

---

## text_analise

Комплексный анализ текстовых данных: TF-IDF важные слова, частые биграммы, визуализация.

### Описание

Скрипт выполняет анализ текстов по эмоциям:

1. **Статистика текстов** - среднее количество слов/символов, примеры
2. **TF-IDF важные слова** - топ-K слов для каждой эмоции с весами
3. **Частые биграммы** - типичные словосочетания из двух слов
4. **Сравнение словарей** - общие и уникальные слова между эмоциями
5. **Визуализация** - bar charts для слов и биграмм
6. **Облака слов** (опционально) - word clouds для каждой эмоции

### Использование

```bash
# Базовый анализ
poetry run python dusha/my_experiments/data_analise/text_analise.py

# С сохранением графиков
poetry run python dusha/my_experiments/data_analise/text_analise.py --save-plots

# Топ-20 слов вместо 10
poetry run python dusha/my_experiments/data_analise/text_analise.py --top-k 20

# Анализ конкретного датасета
poetry run python dusha/my_experiments/data_analise/text_analise.py \
    --dataset combine_balanced_test_small \
    --top-k 15 \
    --save-plots
```

### Параметры

- `--dataset` - Название датасета (по умолчанию: `combine_balanced_train_small`)
- `--top-k` - Количество топ-слов/биграмм для каждой эмоции (по умолчанию: 10)
- `--save-plots` - Сохранять графики в папку `visualizations/`

### Выходные файлы

При использовании `--save-plots`:

1. **top_words.png** - Топ-слова для каждой эмоции (bar charts)
2. **top_bigrams.png** - Топ-биграммы для каждой эмоции
3. **wordclouds.png** - Облака слов (если установлен wordcloud)

### Примеры результатов

#### Важные слова по эмоциям:

**ANGRY:**
- ты, не, что, блядь, тебя, нахуй, меня, мне, тупая, это

**SAD:**
- не, включи, фильм, на, что, как, мне, меня, про, ты

**NEUTRAL:**
- как, на, что, включи, фильм, ты, песня, не, смотреть, кто

**POSITIVE:**
- ты, салют, спасибо, тебя, очень, джой, меня, да, песня, мне

#### Частые биграммы:

**ANGRY:**
- "что ты", "ты что", "ты не", "что ли", "потому что"

**SAD:**
- "не хочу", "что делать", "ты не", "две тысячи", "на ютубе"

**NEUTRAL:**
- "что такое", "две тысячи", "кто такой", "что ты", "как сделать"

**POSITIVE:**
- "привет привет", "да да", "очень очень", "лайк лайк", "джой ты"

### Интерпретация результатов

**Уникальные слова** показывают характерные признаки эмоции:
- `angry`: мат, агрессивные слова
- `positive`: благодарности, позитивные эмоции
- `neutral`: вопросы, команды

**Общие слова** между эмоциями указывают на сложность разделения:
- Местоимения (ты, меня, мне) присутствуют везде
- Нужны контекстные признаки для различения

**Биграммы** дают больше контекста:
- "не хочу" vs "очень очень" - явные эмоциональные маркеры
- Повторения ("да да") характерны для positive

## audio_analise

Комплексный анализ аудио признаков: mel-спектрограммы, MFCC, PCA и t-SNE визуализация.

### Описание

Скрипт выполняет глубокий анализ mel-спектрограмм по эмоциям:

1. **Усредненные mel-спектрограммы** - визуализация типичных паттернов для каждой эмоции
2. **Средние значения MFCC** - анализ мел-частотных кепстральных коэффициентов
3. **PCA визуализация** - линейное снижение размерности для понимания основных компонент
4. **t-SNE визуализация** - нелинейное снижение размерности для выявления кластеров

### Использование

```bash
# Базовый запуск (анализ всех данных, без сохранения графиков)
poetry run python dusha/my_experiments/data_analise/audio_analise.py

# Анализ с ограничением по количеству сэмплов (быстрее)
poetry run python dusha/my_experiments/data_analise/audio_analise.py --max-samples 500

# Сохранение графиков на диск
poetry run python dusha/my_experiments/data_analise/audio_analise.py --save-plots

# Анализ конкретного датасета
poetry run python dusha/my_experiments/data_analise/audio_analise.py --dataset combine_balanced_test_small

# Полный анализ с сохранением
poetry run python dusha/my_experiments/data_analise/audio_analise.py \
    --dataset combine_balanced_train_small \
    --max-samples 1000 \
    --save-plots
```

### Параметры

- `--dataset` - Название датасета (по умолчанию: `combine_balanced_train_small`)
  - Доступные: `combine_balanced_train_small`, `combine_balanced_test_small`, и др.
- `--max-samples` - Максимальное количество сэмплов для анализа (по умолчанию: все)
  - Рекомендуется для быстрого анализа: 100-500 сэмплов
  - Для полного анализа: не указывайте параметр
- `--save-plots` - Сохранять графики в папку `visualizations/`

### Выходные файлы

При использовании `--save-plots` графики сохраняются в `data_analise/visualizations/`:

1. **avg_mel_spectrograms.png** - Усредненные mel-спектрограммы по 4 эмоциям
2. **avg_mfcc.png** - Средние значения MFCC коэффициентов
3. **pca_visualization.png** - PCA проекция в 2D пространство
4. **tsne_visualization.png** - t-SNE визуализация кластеров

### Анализируемые метрики

#### 1. Mel-спектрограммы

Усредненные mel-спектрограммы показывают типичные частотно-временные паттерны для каждой эмоции:
- **Angry**: Часто характеризуется высокими частотами и большой энергией
- **Sad**: Низкие частоты, меньшая энергия
- **Neutral**: Средние значения
- **Positive**: Вариативные паттерны с акцентом на средние частоты

#### 2. MFCC (Mel-Frequency Cepstral Coefficients)

13 коэффициентов, представляющих спектральную огибающую:
- **MFCC 0-1**: Общая энергия и тембр
- **MFCC 2-6**: Основные формантные характеристики
- **MFCC 7-12**: Детали артикуляции

#### 3. PCA (Principal Component Analysis)

Линейное снижение размерности:
- Показывает основные направления вариативности в данных
- Процент объясненной дисперсии указывает на информативность компонент
- Хорошая разделимость классов свидетельствует о качестве признаков

#### 4. t-SNE (t-Distributed Stochastic Neighbor Embedding)

Нелинейное снижение размерности:
- Лучше сохраняет локальную структуру данных
- Выявляет естественные кластеры в данных
- Компактные кластеры указывают на хорошую разделимость эмоций

### Интерпретация результатов

**Хорошие признаки для классификации:**
- Четкая визуальная разница в усредненных спектрограммах
- Непересекающиеся области в PCA визуализации
- Отдельные компактные кластеры в t-SNE

**Проблемные случаи:**
- Перекрывающиеся области в PCA/t-SNE → сложность разделения эмоций
- Похожие паттерны в MFCC → нужны дополнительные признаки
- Низкий процент объясненной дисперсии в PCA → высокая размерность данных


```bash

```

**Графики не отображаются**

Используйте флаг `--save-plots` для сохранения в файлы.

## Сравнение скриптов

| Скрипт | Анализирует | Время | Визуализации |
|--------|-------------|-------|--------------|
| text_analise.py | Тексты (слова, биграммы) | ~10 сек | 2-3 графика |
| audio_analise.py | Аудио (mel, MFCC) | ~5-30 мин | 4 графика |
| main_statistic.py | Статистика датасета | ~1 мин | Опционально |

## Рекомендуемый workflow

1. **Быстрый обзор**: `main_statistic.py` → статистика датасета
2. **Текстовый анализ**: `text_analise.py` → понимание текстовых паттернов
3. **Аудио анализ**: `audio_analise.py --max-samples 500` → предварительный анализ
4. **Полный анализ**: оба скрипта с `--save-plots` на полных данных
